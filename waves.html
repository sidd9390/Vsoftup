Effortless cheque truncation

Waves is an advanced Cheque Truncation System (CTS) for financial institutions, streamlining cheque processing with features like outward, inward, and returns processing. It includes AI for CAR LAR capture and rejects fixing, along with modules for data entry, verification, remote deposit capture, web branch capture, merchant capture, positive pay, and DEM data exchange for NPCI file transfer.

Transforming cheque truncation

Waves is a powerful Cheque truncation system that revolutionizes cheque processing, enabling financial institutions to optimize their operations. Waves automates the capture, verification, and truncation of cheques, improving efficiency, accuracy, and fraud detection, while providing a seamless integration with your existing CBS (Core Banking System) for real-time account information and signature verification.

Key features and benefits

Outward cheque processing

Simplify the processing of outward cheques with Waves. Capture and validate cheque images, perform data entry, and process the cheques seamlessly, ensuring faster clearing and settlement.

Inward cheque processing

Efficiently handle inward cheques with Waves. Capture cheque images, perform data entry and correction, and seamlessly integrate with your CBS to access account details, status, and signatures, enabling accurate and quick processing.

Key features and benefits

Inward returns

Streamline the processing of inward return cheques with Waves. Automatically identify and process return cheques, facilitating quick reconciliation and minimizing delays in the clearing process.

Outward returns

Manage outward return cheques effortlessly with Waves. Automate the processing and reconciliation of return cheques, enabling faster resolution and minimizing errors in the return process.

Key features and benefits

AI-based CAR LAR capture

Leverage advanced artificial intelligence (AI) capabilities to automatically capture and extract CAR (Courtesy Amount Recognition) and LAR (Legal Amount Recognition) information from cheque images. Enhance accuracy and efficiency in cheque processing.

AI to fix rejects

Utilize AI-driven functionality to automatically identify and rectify rejected cheques. Minimize manual intervention and streamline the exception handling process, ensuring smooth cheque processing operations.

Key features and benefits

Data entry and data correction

Waves provides comprehensive modules for data entry and correction, enabling accurate and efficient entry of cheque information. Correct any inaccuracies and ensure data integrity.

L1 and L2 verification

Waves offers robust L1 (Level 1) and L2 (Level 2) verification modules to ensure the accuracy and authenticity of cheque details. Verify signatures, account details, and other critical information for enhanced fraud detection.

Key modules

Remote deposit capture

Waves enables remote deposit capture, allowing customers to conveniently scan and submit cheques for deposit remotely. Enhance customer convenience and accelerate cheque processing.

Web branch capture

Empower branch staff with Waves' web branch capture module. Capture cheque images, perform data entry, and seamlessly integrate with your CBS for real-time information, improving operational efficiency.

Key modules

Merchant capture

Waves facilitates cheque capture at merchant locations, streamlining the processing, reducing transportation costs, and improving cash flow for merchants.

Positive pay

Implement positive pay functionality with Waves. Match issued cheque details with presented cheques to detect and prevent cheque fraud, minimizing the risk of unauthorized transactions.

Key modules

DEM Data exchange module

Waves seamlessly integrates with the DEM data exchange module for secure file transfer with the National Payments Corporation of India (NPCI), ensuring compliance and efficient data exchange.
